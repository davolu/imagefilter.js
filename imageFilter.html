imageFilter.js

About

imageFilter.js is a simple libary that can be used to add effects to html images on the fly with ease....

How to use it

imageFilter.js is very easy to use. It doesnt require any external libary or any css file. All thats needed is the imageFilter.js libary.
Firstly, include the libary in the <head> as in:

<script src="imageFilter.js"> </script>

Now, to use the effect, just add the effect class to the img element as in:

<img src="davo.jpg" class=" img-inverted" />

The above will give the image an inverted effect.

inverted image (from class img-inverted)

Other class values are: 

img-inverted 
img-grayscale 
img-brightness(value)  integer between 1-20
img-contrast(value)    integer between 1-20
img-threshold(value)    integer between 1-20
img-blur 
img-sharpen 
img-edge 
img-reflect 
img-sobel  
img-rgb(r,g,b)  
img-rgba(r,g,b,a)    
img-conv(a,b,c,d,e,f,g,h,i)  3x3 kernel...  a-i are integers (positive and negative)
